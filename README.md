# Node Template

![License](https://img.shields.io/github/license/ja1den/node-template)

> A template for [Node.js](https://nodejs.org/en/) projects.

## Usage

Run the application with [nodemon](https://nodemon.io/).

```sh
npm start
```

Start the app in production mode.

```sh
npm run prod
```
